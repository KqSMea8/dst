Measuring a Daemon Process

Here are several options for setting the path to the coverage file for a daemon process.

  • Modify the script that launches the daemon so that it first sets the COVFILE environment variable.
  • Rename your daemon executable and replace it with a shell script that first sets COVFILE and then launches the real daemon.
  • Use the cov_file function to set the path from inside the daemon.
