covlink - BullseyeCoverage Link 

covlink [options] link ...

DESCRIPTION

covlink intercepts link commands for the compilers listed below and modifies the command to work with the BullseyeCoverage run-time library.

  • CodeWarrior for Windows
  • Intel C++
  • Microsoft C++

Actions taken by covlink include:

  • adding the BullseyeCoverage run-time library appropriate for the target architecture and run-time environment, unless the command already contains a library named libcov-*.
  • adding required system libraries
  • intercepting the entry point if the executable is a DLL
  • working around known linker problems
  • dealing with unusual circumstances such as DLLs containing no code

OPTIONS

-h--help Display a summary of options.
-q--quiet Suppress warnings and the copyright message.
-v--verbose Show informational messages.
--no-lib Do not automatically link with the BullseyeCoverage run-time library.
--no-banner Suppress startup banner

ENVIRONMENT VARIABLES

BULLSEYE_LIB Specifies a location to find the BullseyeCoverage run-time libraries rather than BullseyeCoverage/lib.
COVLINKCFGConfiguration file. See FILES.

FILES

If the files listed below exist, they contain initial options separated by white space. Within such files, $NAME is replaced by the value of environment variable NAME.

BullseyeCoverage/bin/covlink.cfg
home-directory/covlink.cfg
