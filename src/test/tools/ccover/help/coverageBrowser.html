Coverage Browser

Introduction

Coverage Browser is a graphical program that provides easy access to coverage results, control of region selection, the Coverage Build setting, and other build options.

Coverage Browser is supplied in the distributions of almost all operating systems. Coverage Browser can display coverage files created on any platform. The coverage file format does not vary between operating systems.

On Unix, settings are stored in the file $HOME/.BullseyeCoverage. On Windows, settings are stored in the registry.

Except where noted, the information here applies to the Coverage Browser on all platforms.

Launching on Windows

Open the Start menu and choose Programs, then BullseyeCoverage. Or, launch coverageBrowser.exe directly.

Launching on Unix

Run the program BullseyeCoverage/bin/CoverageBrowser. On MacOS, you can also double-click the application icon located in the BullseyeCoverage directory.

Summary Display

Hierarchical navigation of directories, sources, classes, and functions.

First, sort by function coverage to find large areas of code with no coverage at all. Later, sort by condition/decision coverage to narrow your focus with higher accuracy.

Coverage totals.

Coverage Build status.

Source Code Display

Select a function in the left pane to see coverage measurements at the source code level.

Move to the next or previous line that does not have full coverage using toolbar buttons or F4 and Shift+F4, respectively.

Hover the mouse cursor over a measurement to see an explanation of the coverage events in plain English.

Finding Regions with Coverage Criteria

You can create custom queries to quickly find all your regions of code that do not meet criteria specific to your organization's development process. Use the menu command View Queries. Then, use View Query New Query... to enter the criteria for the regions you want to find.

Option Dialog Help

Menu command Tools Options... shows the Options dialog. To see help for the individual options, right-click on them, then select the command What's This?.
