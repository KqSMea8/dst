File Lock Error

SYMPTOMS

The error below occurs with a coverage file or license manager file located on an NFS share. 

File lock error

RESOLUTION

The list below contains known configurations and solutions for those configurations.

  • If your NFS client is HP-UX and your server is Linux, add the insecure option to /etc/exports on the NFS server. For example: 
    /projects 192.168.1.0/255.255.255.0(insecure,rw)
