LynxOS 4

With LynxOS, use the BullseyeCoverage Linux x86 distribution. After extracting the distribution, rebuild the run-time library as shown below. 

cd BullseyeCoverage/run
rm ../lib/*
make

If you do not rebuild the run-time, the symptom below occurs. 

libcov.a(libcov.o): undefined reference to `__errno_location'
