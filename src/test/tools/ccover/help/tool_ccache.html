ccache and distcc

You can use BullseyeCoverage with ccache or distcc when they are invoked by a symbolic link using the name of the compiler. BullseyeCoverage cannot be used with these tools when they are used directly as the compiler, nor when they prefix the compiler command. 

CC=ccache       # No
CC=ccache g++   # No
CC=g++          # Yes

$ ls -l /opt/ccache
lrwxrwxrwx   1 user     group         11 Apr 23 17:05 gcc -> /usr/local/bin/ccache
lrwxrwxrwx   1 user     group         11 Apr 23 17:05 g++ -> /usr/local/bin/ccache

Make sure the PATH environment variable contains BullseyeCoverage/bin before ccache and distcc. For example: 

PATH=/opt/BullseyeCoverage/bin:/opt/ccache:/opt/distcc:$PATH
