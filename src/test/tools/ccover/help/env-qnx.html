QNX

Momentics Windows Host

The Momentics IDE cannot be configured to build with BullseyeCoverage. However, you can build from the command prompt using a makefile as described below. This documentation applies to QNX Software Development Platform 6.3.2.

Build Run-Time Library

Use the commands below to build the BullseyeCoverage run-time library. Modify the compile command to use the same compile options as your project. 

cd BullseyeCoverage/run
qcc -c -O -o libcov-qnx.o libcov-posix.c
qcc -A ../lib/libcov-qnx.a libcov-qnx.o

Build Project

In your project directory, run the commands below. 

make clean
make QCONF_OVERRIDE=BullseyeCoverage/etc/qnx-qconf-override.mk
