MetaWare Development Toolkit for ARC Processors

This product is not officially supported as of November 2009. Although we will make an effort to resolve issues with this product, we cannot guarantee a successful outcome.

This information applies to version 8.2.

Build Run-Time Library

Use the commands below to build the BullseyeCoverage run-time library. Modify the compile command to use the same compile options as your project. 

cd %ProgramFiles%\BullseyeCoverage\run
hcac -c -a7 libcov-arc.c
arac -r ../lib/libcov-arc.a libcov-arc.o

Configure Project

Follow these steps to configure your project.

  1. Use the IDE menu command Project Properties.
  2. Select C/C++ Build: Settings.
  3. Select the C++ compiler.
  4. Change the Command setting to the quoted full path of the BullseyeCoverage interceptor, "c:/Program Files/BullseyeCoverage/bin/hcac".
  5. Select the linker and repeat step 4.
