Samples

Prebuilt Coverage File

A prebuilt sample coverage file is provided in the BullseyeCoverage directory sample. You can use this file to see how the BullseyeCoverage reporting tools work. Open the file with the Coverage Browser or command line programs.

Building with Visual C++

  1. Open sample.dsw
  2. Choose command Tools Enable/Disable Coverage Build
  3. Choose command Build Rebuild.
  4. Press F5 to run.
  5. From the Start menu, choose Programs BullseyeCoverage.

Building with the Makefile

You can use the supplied Makefile to build the samples at a shell prompt. 

cov01 -1
make run
cov01 -0
